## üìã Interview Checklist

### Before the Interview
- [ ] Review all study materials
- [ ] Practice coding problems (at least 20-30)
- [ ] Prepare 5-10 behavioral stories using STAR method
- [ ] Research Learntube AI thoroughly
- [ ] Prepare questions to ask interviewers
- [ ] Test your setup (if remote interview)
- [ ] Get good sleep the night before

### During the Interview
- [ ] Listen carefully to questions
- [ ] Ask clarifying questions
- [ ] Think out loud
- [ ] Start with brute force, then optimize
- [ ] Consider edge cases
- [ ] Write clean, readable code
- [ ] Test your solution with examples

### After the Interview
- [ ] Send thank-you email
- [ ] Reflect on what went well
- [ ] Note areas for improvement
- [ ] Follow up if needed

## üí° Tips for Success

1. **Practice Coding Daily**: Solve at least 2-3 problems per day
2. **Understand, Don't Memorize**: Focus on understanding concepts
3. **Explain Your Thinking**: Practice explaining solutions out loud
4. **Time Yourself**: Practice solving problems within time limits
5. **Review Common Patterns**: Two pointers, sliding window, etc.
6. **Stay Calm**: Take deep breaths, think step by step
7. **Ask Questions**: Clarify requirements before coding
